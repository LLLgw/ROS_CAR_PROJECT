# ROS 小车固件

这是一个基于 STM32F4 和 FreeRTOS 的 ROS 小车底盘固件工程，集成了 ROS 串口通信、PS2 手柄控制、四电机闭环调速、电池电压检测、OLED 状态显示和急停保护等功能。

## 项目简介

本项目用于四电机差速底盘小车的嵌入式控制。固件运行在 STM32F4 系列 MCU 上，整体采用 FreeRTOS 进行任务调度，主要负责以下功能：

* 底盘运动控制
* ROS 串口通信
* PS2 手柄输入解析
* 电池电压采样
* OLED 状态显示

项目底层基于 STM32 Standard Peripheral Library，并通过 `USART3` 实现自定义 ROS 串口协议通信。

## 功能特性

* 四路电机 PWM 驱动
* 四路编码器测速反馈
* 基于 PI 的闭环速度控制
* 双控制源切换：

  * ROS 上位机控制
  * PS2 手柄控制
* 控制源在线状态判断与切换
* 电池电压检测与低电量标志
* 急停输入保护
* OLED 信息显示
* 基于 FreeRTOS 的多任务调度

## 软件架构

固件从 `main()` 启动，依次完成板级初始化、任务创建，并启动 FreeRTOS 调度器。

### FreeRTOS 任务

* `Control_Task`

  * 周期：`10 ms`
  * 功能：执行底盘控制、速度反馈计算和电机输出
* `RosComm_Task`

  * 周期：`5 ms`
  * 功能：接收 ROS 速度指令并上报底盘/电池状态
* `PS2_Task`

  * 周期：`1 ms`
  * 功能：轮询 USB 手柄状态并生成本地控制指令
* `Battery_Task`

  * 周期：`100 ms`
  * 功能：采样电池电压并更新电池状态
* `Display_Task`

  * 周期：`200 ms`
  * 功能：刷新 OLED 显示内容

## 控制逻辑

底盘支持两个控制源：

* `ROS`
* `PS2`

当前选中的控制源保存在全局状态中，由底盘控制任务决定实际采用哪一路速度指令。如果选中的控制源离线或数据无效，底盘会自动停车。

### 当前源码中的运动限制

* 最大线速度 `vx`：`0.45 m/s`
* 最大角速度 `wz`：`1.20 rad/s`
* 最大车轮目标速度：`0.55 m/s`

当前控制环还包含以下机制：

* 速度渐变控制
* 弧线转向限制
* 输出限幅
* 停车稳定判定

## ROS 串口协议

通信使用 `USART3`，波特率为 `115200`。

### 帧头帧尾

* 帧头：`0xAB`
* 帧尾：`0xAC`

### 接收帧 RX

长度：`17 字节`

主要内容：

* 帧类型或保留字节
* `vx_mps`，类型为 `float`
* `vy_mps`，类型为 `float`
* `wz_radps`，类型为 `float`
* XOR 校验

### 发送状态帧 TX

长度：`17 字节`

上报内容：

* `vx_mps`
* `vy_mps`
* `wz_radps`
* 状态标志位

### 电池上报帧

长度：`13 字节`

上报内容：

* 电池电压 `mV`
* 电池电流 `mA`
* 电量百分比
* 电池状态标志

### ROS 超时机制

如果 `500 ms` 内没有收到有效 ROS 速度指令，固件会将 ROS 状态置为离线，并清空当前 ROS 控制指令。

## PS2 手柄控制说明

PS2/手柄输入通过 USB 模块处理。

根据当前源码，控制行为如下：

* 按下 `START` 解锁手柄控制
* 按下 `X` 在 `PS2` 和 `ROS` 控制源之间切换
* 按下 `LB` 降低速度上限
* 按下 `RB` 提高速度上限
* 摇杆和扳机用于前进/后退控制
* 右摇杆用于转向控制

速度上限范围：

* 最小值：`0.10 m/s`
* 默认值：`0.25 m/s`
* 最大值：`0.45 m/s`

## 电池管理

电池电压通过 `ADC1` 采样，输入引脚如下：

* `PB0`
* ADC 通道 `8`

当前源码中的电池参数设定：

* 分压系数：`11`
* 满电电压：`25.2 V`
* 空电电压：`19.8 V`
* 低电阈值：`20.4 V`

固件会更新以下状态：

* 电池电压
* 电量百分比
* 低电量标志

注意：当前源码中 `BAT_STOP_ENABLE = 0`，即低电自动停车功能默认关闭。

## 安全保护

固件当前包含两类停车保护：

* 急停输入
* 可选的低电保护停车

急停输入配置如下：

* GPIO 引脚：`PE4`
* 触发方式：低电平有效

当保护条件触发时，固件会：

* 停止所有电机
* 清空车轮控制环状态
* 清零最终输出指令
* 将运动模式切换为停止

## 硬件说明

该工程面向 `STM32F4` 平台，根据代码，当前使用了以下外设：

* `TIM8`：电机 PWM 输出
* `TIM2`、`TIM3`、`TIM4`、`TIM5`：编码器计数
* `USART3`：ROS 串口通信
* `ADC1`：电池电压采样
* GPIO 模拟时序：OLED 显示驱动
* USB 相关模块：手柄输入

如果你要移植到其他硬件平台，建议先逐一检查 `BSP/` 目录中的引脚定义和外设配置。

## 编译说明

当前压缩包中包含源码、FreeRTOS、CMSIS 和 STM32 标准外设库，没有完整的 IDE 工程文件，例：

* Keil `.uvprojx`
* STM32CubeIDE `.project` / `.cproject`
* IAR 工程文件

因此在编译前，你通常需要：

1. 在你的 IDE 中新建一个 STM32F4 工程
2. 将本仓库中的源码和头文件路径加入工程
3. 按实际 MCU 型号配置启动文件和链接脚本
4. 检查 FreeRTOS 中断优先级配置是否匹配
5. 按照实际 PCB 检查 BSP 中的引脚映射

## 快速开始

1. 将源码导入新的 STM32F4 工程
2. 确认目标芯片与启动文件、外设配置一致
3. 检查电机、编码器、OLED、串口、USB 和电池采样引脚定义
4. 将 ROS 上位机连接到 `USART3`，波特率设为 `115200`
5. 烧录后重点验证以下内容：

   * 编码器反馈是否正常
   * 电机转向是否正确
   * 急停是否有效
   * PS2 与 ROS 控制切换是否正常
   * ROS 超时停车逻辑是否正常
